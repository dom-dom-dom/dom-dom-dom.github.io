Animated CSS TARDIS

by Dominic M. Liddell

This is an animated version of "CSS TARDIS" by John Galanti. I originally decided to replicate this to better understand how it was created and because I love Doctor Who. I found the CSS TARDIS because I was looking for the perfect shade of TARDIS blue for my blog.

I also wanted to add my own stank to it, so I looked for ways to add to the project in a way that made sense. Initially I was going to add a blue line down the middle of the lamp like you see in some versions, then I thought about it more and came to the conclusion that the blue line is only showing when it rotates.

I ended up not adding the blue line because I spent a lot of time on getting the letters to face the right way during the rotation and not be backwards on some spins. I did get the light to flash as it rotates. This may not be 100% accurate, but I felt like it was a nice effect. Part of the magic of the TARDIS is the sound it makes, so I added a short audio of it landing.
